# CineLog — Tu universo audiovisual 🎬

App personal para seguimiento de series y películas.

## Cómo subir a GitHub Pages (5 minutos)

### 1. Crea una cuenta en GitHub
Ve a [github.com](https://github.com) y crea una cuenta gratuita.

### 2. Crea un repositorio nuevo
- Pulsa el botón **"New"** (verde, arriba a la izquierda)
- Nombre: `cinelog` (o el que quieras)
- Marca **"Public"**
- Pulsa **"Create repository"**

### 3. Sube los archivos
En la página del repositorio vacío:
- Pulsa **"uploading an existing file"**
- Arrastra TODA la carpeta `cinelog-pwa` (o selecciona todos los archivos)
  - `index.html`
  - `manifest.json`
  - `sw.js`
  - `icons/icon-192.png`
  - `icons/icon-512.png`
- Pulsa **"Commit changes"**

### 4. Activa GitHub Pages
- Ve a **Settings** (pestaña del repositorio)
- Menú lateral → **Pages**
- En "Branch": selecciona `main` y carpeta `/root`
- Pulsa **Save**
- Espera 2-3 minutos

### 5. Tu URL
Será algo como: `https://TU_USUARIO.github.io/cinelog`

### 6. Instalar en Android (Pixel)
1. Abre la URL en **Chrome**
2. Espera a que cargue completamente
3. Chrome mostrará un banner **"Añadir CineLog a la pantalla de inicio"**
   - Si no aparece: menú ⋮ → "Añadir a pantalla de inicio"
4. ¡Listo! Aparece como app nativa en tu Pixel

## Características
- 📺 Seguimiento de series con temporadas y capítulos
- 🎬 Registro de películas
- 🔍 Búsqueda automática desde TMDB/IMDb
- ⭐ Sistema de puntuaciones con medias automáticas
- 💾 Datos guardados localmente en el navegador
- 📱 Diseño iOS-style, funciona offline
